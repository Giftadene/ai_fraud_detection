Download & Installation Guide

Run the system locally on any Windows machine. No coding experience required.

bolt Quick Start (Windows)

  1. Download Python from python.org/downloads (check "Add Python to PATH" during installation).
  2. Download the project as a ZIP file using the button below and extract it to a folder.
  3. Double-click start.bat — it will automatically install dependencies and launch the app.
  4. Your browser will open at http://127.0.0.1:5000.

menu_book Detailed Installation Steps

Step 1: Install Python

Download and install Python 3.10 or later from python.org. Important: Check the box "Add Python to PATH" during installation.

Step 2: Download & Extract

Download the project ZIP file and extract it to any folder (e.g., C:\FraudGuardAI).

Step 3: Run the App

Double-click start.bat. This script will:

  • Verify Python is installed
  • Create a virtual environment
  • Install all required dependencies (Flask, Pandas, NumPy, Scikit-learn, XGBoost)
  • Launch the application
  • Open your browser to the login page

The first run may take 1-2 minutes as it installs dependencies and trains the machine learning model.

Step 4: Login

Use the credentials above to log in as an Administrator or Fraud Analyst and explore the system.

terminal Manual Command Line Setup

Alternatively, run these commands in the project folder:

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
set FRAUDGUARD_ADMIN_PW=admin123
set FRAUDGUARD_ANALYST_PW=analyst123
python app.py

Then open http://127.0.0.1:5000/login in your browser.
